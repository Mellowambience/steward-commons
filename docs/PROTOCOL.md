# Protocol Specification

## Core objects

- Project
- Actor
- Decision
- Proposal
- Evidence
- Outcome
- Reflection

## Decision lifecycle

1. Identify decision need.
2. Define context.
3. Generate proposals.
4. Attach evidence.
5. Record objections.
6. Select decision.
7. Define review window.
8. Observe outcomes.
9. Add reflection.
10. Update future reasoning.

## Compatibility goals

A Steward Commons object should be serializable as JSON, exportable as a graph, and readable without proprietary software.

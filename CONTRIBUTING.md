# Contributing

Steward Commons welcomes contributions that improve the protocol, documentation, schemas, examples, and reference implementations.

## Principles

- Preserve reasoning, not just conclusions.
- Make assumptions visible.
- Prefer interoperability over lock-in.
- Separate protocol from product.
- Build for future reflection.

## Contribution types

- Protocol improvements
- Schema updates
- Example decision graphs
- Governance proposals
- Reference implementation work
- Research notes
